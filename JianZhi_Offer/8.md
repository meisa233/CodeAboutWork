### 题目描述：
一只青蛙一次可以跳上1级台阶，也可以跳上2级。求该青蛙跳上一个n级的台阶总共有多少种跳法（先后次序不同算不同的结果）。
### 答案：
```C++
class Solution {
public:
    int jumpFloor(int number) {
        int Count = 0;
        if(number==1){
            return 1;
        }else if(number==2){
            return 2;
        }else{
            return jumpFloor(number-1)+jumpFloor(number-2);#递归：第一次跳跃是1步和第以次跳跃是2步的两种情况
        }
    }
};
