### 题目：
请实现一个函数，将一个字符串中的每个空格替换成“%20”。例如，当字符串为We Are Happy.则经过替换之后的字符串为We%20Are%20Happy。
### 答案：
```C++
class Solution {
public:
	void replaceSpace(char *str,int length) {
		char kongge = ' ';

		int i = 0;
		while (str[i] != '\0') {
			if (str[i] == kongge) {
				length += 2;
				for (int j = length - 1; j > i + 2; j--) {
					str[j] = str[j - 2];
				}
				str[i] = '%';
				str[i + 1] = '2';
				str[i + 2] = '0';
			}
			else {
				i++;
			}
		}
	}
};
```
### Note：
注意这是一个动态字符串，每次替换掉空格后需要将空格后的子字符串依次后移。
